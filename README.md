## ⚙️ Instalación

Para instalarlo en la computadora, debes seguir estos pasos:

1. Clona el repositorio

```bash
https://github.com/ImLauraGS/pressuposts.git
``` 
2. Instalar dependencias.

```bash
npm install
``` 
3. Correr Json-Server

```bash
npm run runApi
``` 
4. Correr aplicación.

```bash
npm run dev
```
